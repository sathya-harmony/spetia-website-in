Useful layout lessons for Studio Spetia:

- Start with an image-led first impression, then let restrained copy arrive after the photograph has done its work.
- Keep navigation quiet and persistent. The primary enquiry path should be visible without adding a second CTA style.
- Use large photographic breaks between prose sections so the page feels like a sequence of built moments.
- Let project pages run long. A single oversized image, then project details, then another image is stronger than a dense gallery grid.
- Repeat the contact invitation after the visual story rather than interrupting every section.
- Keep typographic contrast high: display serif for headings, quiet sans for body and controls.
